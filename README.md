# Nkom
aptやnpmのような感覚でプラグインを管理出来るツール
# 使い方
プラグインのインストール
```
nkom install example
```
プラグインの削除(レジストリにない場合はプラグインの名前で削除可能)
```
nkom remove example
```
```install```などの代わりに```i``` or ```add```などが使用出来ます。
# レジストリの管理
